## 思路
- p 的长度是固定的 → 窗口大小固定为 p.length()
- 用两个数组（或哈希表）分别记录窗口内和 p 中每个字符出现的次数
- 窗口滑动时：
右边进来一个字符 → 窗口计数 +1
左边出去一个字符 → 窗口计数 -1

- 只要当前窗口的计数数组 == p 的计数数组，就说明找到了一个异位词

## 代码部分
```c++
class Solution {
public:
    vector<int> findAnagrams(string s, string p) {
        if (s.size()<p.size()) return {};

        vector<int> ans;
        int n=s.size(), m=p.size();

        //统计p中每个字符出现次数
        vector<int> p_cnt(26,0);
        for (char c:p){
            p_cnt[c-'a']++;
        }

        //统计当前窗口中每个字符出现次数
        vector<int> win_cnt(26,0);

        for (int i =0;i<m;i++){
            win_cnt[s[i]-'a']++;
        }

        if (win_cnt == p_cnt){
            ans.push_back(0);
        }

        for (int right =m;right<n;right++){
            int left = right-m;

            win_cnt[s[right]-'a']++;
            win_cnt[s[left]-'a']--;

            if (win_cnt == p_cnt){
                ans.push_back(left+1); //注意此时检测的子串其实是[left+1,right],因为把left的统计量去除了，而右边是包含的
            }
        }

        return ans;
     }
};
```
